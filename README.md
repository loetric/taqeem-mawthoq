# تقييم موثوق - منصة تقييم الأماكن

منصة تقييم موثوق لتقييم الأماكن في المملكة العربية السعودية مع ضمان نزاهة التقييمات.

## الميزات

- ✅ تقييم الأماكن مع تفاصيل شاملة
- ✅ نظام نقاط الولاء والمكافآت
- ✅ تقييمات الخبراء والموثوقين
- ✅ لوحة تحكم متكاملة لأصحاب الأنشطة
- ✅ إشعارات موجهة
- ✅ تحليلات وإحصائيات
- ✅ تصميم متجاوب مع جميع الأجهزة

## التقنيات المستخدمة

- **Next.js 16** - إطار عمل React
- **TypeScript** - للبرمجة الآمنة
- **Tailwind CSS** - للتصميم
- **Lucide React** - للأيقونات

## التشغيل المحلي

```bash
# تثبيت المكتبات
npm install

# تشغيل السيرفر للتطوير
npm run dev

# بناء المشروع للإنتاج
npm run build

# تشغيل النسخة المبنية
npm start
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح.

## النشر

راجع ملف `DEPLOYMENT.md` للحصول على دليل شامل للنشر.

### النشر السريع على Vercel:

1. ارفع المشروع على GitHub
2. اذهب إلى [Vercel](https://vercel.com)
3. سجل دخول وربط GitHub
4. اضغط "Add New Project" واختر المستودع
5. اضغط "Deploy"

بعد دقائق قليلة، ستحصل على رابط يمكنك فتحه من أي جهاز!

## البنية

```
├── app/              # صفحات Next.js
├── components/       # المكونات المشتركة
├── lib/             # المكتبات والوظائف المساعدة
├── types/           # تعريفات TypeScript
└── public/          # الملفات الثابتة
```

## ملاحظات

- البيانات حالياً محفوظة في `localStorage` (محلية للمتصفح)
- كل مستخدم يرى بياناته فقط
- البيانات لا تنتقل بين الأجهزة
